## Description
This is starter project that will implement ideal of full restaurant backend that contains multiple vendors that tack orders , transaction and so much more it is developed using NestJs with mongoDb
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
